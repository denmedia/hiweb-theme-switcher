== Description ==

= English =

Changing the theme on selected pages and posts

Instructions (after you install and activate this plugin):

1. Go to the list of rules: "Appearance → hiWeb Theme Switcher"
2. Create a new, or edit one of the rules.
3. On these pages or posts the theme will be replaced by the specified in the rule. Enjoy;))


= Русский =

Смена темы сайта на определенных страницах и записях.

Инструкция (после установки и активации данного плагина):

1. Зайдите в список правил: "Оформление → hiWeb Theme Switcher"
2. Создайте новое или отредактируйте одно из правил.
3. На указанных страницах или постах тема будет сменяться на указанную в правиле. Приятного аппетита ;))


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload folder `hiweb-core` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'hiWeb Core' menu in WordPress
3. Go to the list of rules: "Appearance → hiWeb Theme Switcher"
4. Create a new, or edit one of the rules.
5. On these pages or posts the theme will be replaced by the specified in the rule.
